Projects

Most recent work

Car spot detection and counter system

Car spot detection and counter system , is a computer vision project capable of processing video feeds at 30 frames per second (fps). The system is designed to efficiently detect and count cars in real-time, making it a valuable tool for traffic monitoring, parking management, and smart city applications.

GitHub Repository

Car license plate detection

This advanced system not only identifies cars accurately but also extracts license plate information swiftly and efficiently. By integrating YOLOv8 with tracking algorithms like SORT, we enable continuous monitoring and tracking of vehicles across frames, ensuring robust performance in diverse environments. This project showcases the application of cutting-edge computer vision techniques to enhance security, automate surveillance, and streamline traffic management systems effectively.

GitHub Repository

Hand Gesture volume control

Our project utilizes MediaPipe, a versatile framework for building cross-platform AI solutions, to develop a hand gesture-based volume control system. By leveraging real-time hand tracking and gesture recognition capabilities, the system interprets specific hand movements to adjust audio volume dynamically. This innovative approach eliminates the need for physical controls, offering intuitive and hands-free interaction with devices.

GitHub Repository

People Counter using Yolov8

People Counter using YOLOv8 project is a real-time system designed to accurately count people entering and exiting a specified area within a video stream. Leveraging the YOLOv8 deep learning model, OpenCV for video processing, and SORT (Simple Online and Realtime Tracking) for object tracking, this system offers robust performance in dynamic environments such as malls, public spaces, or event venues

GitHub Repository

Coffee machine virtual interface using CV

The Coffee Machine Virtual Interface is an innovative project that uses cutting-edge computer vision technology to create a touchless coffee ordering system. Leveraging Python, OpenCV, and MediaPipe, this interface allows users to select their coffee type, sugar level, and cup size through simple hand gestures. This intuitive system even records the interaction, providing a seamless and modern coffee ordering experience.
